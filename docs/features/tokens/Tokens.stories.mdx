# Tokens

Tokens are Orbiter's reusable values to provide a common language between developers and designers.


> If you are looking for a list of available tokens, please refer to the [Hopper tokens page](https://hopper.workleap.design/tokens/getting-started/introduction).


## Usage

Tokens can be used as [style props](/docs/style-props--page) values:

```jsx
<Div backgroundColor="neutral-weak">...</Div>
```

Or `CSS` variables:

```css
.item {
    background-color: var(--hop-neutral-surface-weak);
}
```
