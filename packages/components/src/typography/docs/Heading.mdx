## Usage

### Default

A default heading will render as a `<div>` element.

<Canvas of={HeadingStories.Default} />

### Heading elements

For most cases, we recommend using HTML content sectionning heading elements instead of a `<div>`. Orbiter help by offering specialized heading components `<H1>`, `<H2>`, `<H3>`, `<H4>`, `<H5>`, `<H6>`.

<Canvas of={HeadingStories.HeadingElements} />

### Size

You can alter the size of an heading component by specifying a `size` prop.

<Canvas of={HeadingStories.Size} />
