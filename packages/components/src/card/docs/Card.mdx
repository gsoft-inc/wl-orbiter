# Card

## Guidelines

### When to use

- To restrain elements to a non-interactive container. Child elements can be interacted with.

### Card vs Tile

- If you need an interactive or navigable container, use a [tile](?path=/story/tile--default-story) component.

A card must have an heading and a content.
