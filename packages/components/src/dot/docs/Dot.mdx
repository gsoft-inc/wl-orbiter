### Color

A dot color can vary.

You can use a color name from our Orbiter design system (e.g. `moss-200`, `--o-ui-accent-6`), an hexadecimal color code (e.g. `#c5cac8`), an `hsl`, an `hsla`, a `rgb` or a `rgba` color code.
