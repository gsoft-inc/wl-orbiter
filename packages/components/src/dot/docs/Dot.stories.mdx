A dot can have a label.

### Color

A dot color can vary.

You can use a color name from our Orbiter design system (e.g. `alias-2`, `--o-ui-accent-6`), an hexadecimal color code (e.g. `#c5cac8`), an `hsl`, an `hsla`, a `rgb` or a `rgba` color code.

<Preview>
    <Story name="color">
        <Inline>
            <Dot color="alias-hard-break">Habitable</Dot>
            <Dot color="hsl(25, 69%, 41%)">Uninhabitable</Dot>
            <Dot color="--o-ui-accent-6">Clean</Dot>
        </Inline>
    </Story>
</Preview>
