## Usage

A main component accepts all the [main HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/main) and [Orbiter styled component props](?path=/docs/styling--page).

```jsx
<body>
    <Header>
        <H1>Walking On The Moon</H1>
    </Header>
    <Main color="alias-tertiary">
        <Paragraph>Walking on the Moon is classic Sting moment, with his band The Police turning out one of their most groovy reggae-inspired hits to the background of space exploration.</Paragraph>
    </Main>
</body>
```
