## Usage

A footer component accepts all the [footer HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/footer) and [Orbiter styled component props](?path=/docs/styling--page).

```jsx
<HtmlArticle>
    <H1>What does an aerospace engineer do?</H1>
    <UL>
        <LI>Research and development</LI>
        <LI>Testing</LI>
        <LI>Production and maintenance</LI>
    </UL>
    <HtmlFooter color="neutral-weak">
        <Paragraph>Â© 2021 Orbiter</Paragraph>
    </HtmlFooter>
</HtmlArticle>
```
